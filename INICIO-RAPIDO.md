# 🚀 INICIO RÁPIDO

Para crear un nuevo proyecto usando este template:

## Windows (PowerShell)

```powershell
# 1. Copiar template
Copy-Item -Path "syp-react-template" -Destination "MI-NUEVO-PROYECTO" -Recurse

# 2. Entrar al proyecto
cd MI-NUEVO-PROYECTO

# 3. Instalar dependencias
npm install

# 4. Iniciar desarrollo
npm run dev
```

## Luego...

1. ✏️ Edita `src/App.jsx` - Cambia título y contenido
2. 🎨 Modifica `src/App.css` - Personaliza colores si necesitas
3. 📦 Instala librerías adicionales: `npm install lo-que-necesites`
4. 🚀 Empieza a programar tu lógica

## Archivos Clave

- `src/App.jsx` → Componente principal (EMPIEZA AQUÍ)
- `src/App.css` → Todos los estilos
- `public/Logo syp.png` → Logo corporativo
- `index.html` → Cambia el título de la pestaña

## Listo! 🎉

Ya tienes el diseño corporativo completo.
Solo agrega tu lógica de negocio.
